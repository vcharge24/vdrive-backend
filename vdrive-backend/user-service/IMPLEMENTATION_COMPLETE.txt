Multi-Tenancy Support:
    - Hierarchical structure: Tenant → Company → Location
    - Tenant-scoped roles and users
    - User inheritance of hierarchy from creator

Authentication & Authorization:

    - HS256 algorithm with jjwt 0.12.6
    - Role-based access control (RBAC)
    - Screen endpoint-based privileges
    - Support for ADMIN and MERCHANT portals

Data Persistence:
    - PostgreSQL with Spring Data JPA
    - UUID primary keys for better distribution

Service Architecture:
